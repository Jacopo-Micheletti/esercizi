# Cifrario di Cesare

Si scriva una funzione che accetta come parametri una stringa e un intero (chiave di cifratura) e che applichi alla stringa il sistema di [cifratura di Cesare](https://it.wikipedia.org/wiki/Cifrario_di_Cesare). Il cifrario di Cesare sposta avanti ogni lettera del messaggio da cifrare di un numero uguale di posizioni nell'alfabeto in base al valore della chiave: se ad esempio il testo da cifrare fosse "arriviamo" e lo spostamento 3, il testo cifrato diventerebbe "duulyldpr". La chiave 0 lascia il testo inalterato.

Si scriva successivamente un programma per testare la correttezza della funzione creata.
