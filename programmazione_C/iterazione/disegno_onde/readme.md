# Disegno onde
Si scriva un programma che dato in input un intero positivo **N** disegni un numero **N** di onde secondo il seguente schema:

### Esempio con N = 1
<pre>
 _
/ \
   \_/
</pre>

### Esempio con N = 2
<pre>
 _     _
/ \   / \
   \_/   \_/
</pre>

### Esempio con N = 3
<pre>
 _     _     _
/ \   / \   / \
   \_/   \_/   \_/
</pre>
